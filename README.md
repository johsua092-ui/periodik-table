# Interactive Periodic Table

High-quality, professional Interactive Periodic Table built with Next.js, Framer Motion, and Tailwind CSS.

## Features
- Complete 118 Elements
- Interactive Bohr Model Visualizer
- Real-time Search & Category Filters
- Educational Quiz Mode
- Study Guide (Bahasa Gaul)

## Deployment
This project is automatically deployed to GitHub Pages via GitHub Actions.
